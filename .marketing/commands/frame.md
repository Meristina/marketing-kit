# /marketing.frame — Phase 0 (Commander)

**Constitution:** re-read `.marketing/memory/constitution.md` (esp. Art. III mirror language,
VI minimal MECE).

`$MISSION` = `$ARGUMENTS` (the mission dir).

## Do
1. **Read in**: `$MISSION/dossier.md` — don't re-ask what's already there.
2. Restate the marketing goal in one sentence; name the **product, audience, market, B2B/B2C,
   stage, and the business objective** you detect.
3. Ask **2–3 questions that change the plan** (objective, target, budget, constraints, timeline,
   success metric), each with a recommended default. **Wait for answers** unless told "go".
4. **Write out**: set `context` in `$MISSION/dossier.md`, record the framing decision and any new
   assumptions (tagged) and open-to-verify items.

## Done when
`context` is set in the Dossier and the plan-changing questions are asked (or answered). Mirror
the user's language.
